# Thieves City - Project Summary

## Overview

**Thieves City** is a 2.5D multiplayer action game built with Godot Engine 3.2.3. Players control a thief in a bustling city environment, competing against 30 AI or online opponents to collect the most loot within a time-limited match.

## Project Information

- **Repository**: https://github.com/elith-bot/Thieves-City
- **Engine**: Godot Engine 3.2.3
- **Language**: GDScript
- **License**: MIT
- **Status**: Early Development (v0.1.0)
- **Platforms**: Web, Android, iOS, Windows, macOS, Linux
